工作区
暂存区

git status
git diff
git add .	 文件修改添加到暂存区；
git commit -m 'a'	把暂存区的所有内容提交到当前分支。
git log 	可以查看提交历史
git reflog	 //查看命令历史


git reset --hard adcs1sa  // 回退到某版本
git reset HEAD .  撤销工作区的修改  即撤销add   命令既可以回退版本，也可以把暂存区的修改回退到工作区。
git checkout -- . 放弃修改 丢弃工作区的修改：

git rm 1.txt 版本库中删除该文件

git remote add origin git@github.com:xxxx.xx.git 	remote  遥远  origin 起源
git push -u origin master 推送远程库  // 了-u参数，Git不但会把本地的master分支内容推送的远程新的master分支，还会把本地的master分支和远程的master分支关联起来
git clone  git@github.com:xxxx.xx.git
